# Команды
- npm run dev: Запуск dev-сервера
- npm run build: Сборка проекта
- npm run lint: Проверка кода

# Стиль кода
- Использовать TypeScript
- Функциональные компоненты React
- Tailwind для стилей
- Максимум 50 строк на компонент

# Workflow
- Перед изменениями: изучи существующий код
- После изменений: проверь в браузере
- Коммить с понятными сообщениями на русском
